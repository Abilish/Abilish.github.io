1 time clock()方法及替代方法

python 3.8开始将移除 clock() 方法，使用 time.perf_counter() 或 time.process_time() 方法替代。

1.1 time.clock()

因为我没有安装3.7版本的python，所以使用了在线工具试了一下 clock() 方法

在线工具网站（提供不同版本的代码测试）：python在线编程

time.clock()函数以浮点数计算的秒数返回当前的CPU时间，用来衡量耗时。

  • UNIX系统：返回“进程时间”，用秒表示的浮点数
  • WIN系统：第一次调用返回进程运行的实际时间，第二次调用
